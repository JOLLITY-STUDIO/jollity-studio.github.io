关于 CHINA JOLLITY STUDIO（华悦工作室）

这是 CHINA JOLLITY STUDIO（华悦工作室）的轻量级静态站点模板，适合作为个人主页、作品集或简单项目站点的起点。易于维护，不依赖构建工具。

站点说明

  • 基于静态文件，直接部署到 GitHub Pages。
  • 使用 Bootstrap 提供响应式布局，保持兼容性与可维护性。
  • 可根据需要扩展为博客或集成第三方表单服务。

技术栈

纯静态 HTML/CSS，Bootstrap CDN（可替换为本地），简单 CSS 变量用于主题色。
